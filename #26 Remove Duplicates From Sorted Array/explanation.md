As I walk through the array, I ask a question: is this element a duplicate? If it isn't I will add it to the hashmap. If an element is a duplicate, I will increment one of my pointers, but keep the other on that position so that when I do find the next unique element, I can copy it there. Since the numbers after k, which represents the number of unique elements in an array, are not considered, we don't need to manipulate or change those elements.
